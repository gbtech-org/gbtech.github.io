Install Highlighted Cursor Theme for XFCE

  1. Download the bDMZ Cursor theme Download
  2. Extract the downloaded archive with the following command:
    cd ~/Downloads
    mkdir bDMZ && mkdir ~/.icons
    # extract the downloaded tar.gz
    sudo tar xzvf *bDMZ*tar.gz -C ~/Downloads/bDMZ --strip-components=1
    # copy the extracted cursor themes to ~/.icons
    sudo mv ~/Downloads/bDMZ/* ~/.icons
  3. Click the XFCE button > Settings > Mouse and Touchpad
  4. Select the Theme tab
  5. Select the desired bDMZ cursor theme
